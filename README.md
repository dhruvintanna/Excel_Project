# Excel Data Analysis Portfolio

Welcome to my Excel projects repository! This repository showcases my ability to clean, analyze, and visualize complex datasets to drive data-driven business decisions.

## 🛠️ Core Skills Demonstrated
* **Data Cleaning & Transformation:** Power Query, Text manipulation (`SUBSTITUTE`, `REPLACE`), `TRIM`, handling missing data (`IFERROR`, `AVERAGEIFS`).
* **Advanced Formulas:** Dynamic arrays, logic functions (`IF`, `AND`), conditional counting/averaging (`COUNTIF`, `AVERAGEIF`).
* **Data Modeling:** Power Pivot, DAX, structured table formatting, and fixing memory/spill errors.
* **Visualization:** Interactive dashboards, Conditional Formatting, and Pivot Tables.
