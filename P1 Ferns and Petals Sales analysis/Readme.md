# 💐 FNP Sales Analysis Dashboard (MS Excel)

## 📌 Project Overview
An interactive, end-to-end sales performance dashboard built entirely in Microsoft Excel analyzing e-commerce transactions for **Ferns N Petals (FNP)**. This dashboard transforms raw transactional data into actionable commercial insights regarding revenue drivers, regional demand, delivery timelines, and customer spending habits.

## 🛠️ Key Technical Skills Demonstrated
* **Data Cleansing & ETL:** Handled missing timestamps, managed currency formatting (INR ₹), and extracted dynamic date components using advanced Power Query / Excel text logic.
* **Advanced Data Modeling:** Formatted data structures to cleanly support multi-parameter tracking across various time periods, locations, and product segments.
* **Time-Series Analysis & Trends:** Evaluated delivery efficiency metrics and mapped diurnal sales cycles using continuous line graphs.
* **Interactive UI/UX Design:** Integrated coordinated multi-slicer panels (`Order_Date`, `Delivery_Date`, and `Occasion`) with synchronized timelines to enable deep-dive filtering.

---

## 📊 Core Business Insights & KPIs
* **Topline Metrics:** Tracked overall performance utilizing high-level KPI cards for **Total Orders (126)**, **Total Revenue (₹5,86,176.00)**, **Avg. Delivery Days (~5.7 days)**, and **Avg. Customer Spending (₹4,652.19)**.
* **Occasion-Based Drivers:** Identified *Anniversary* and *Raksha Bandhan* as the primary operational revenue engines.
* **Category Penetration:** Maintained product grouping performance data showcasing *Soft Toys* as the dominant revenue category.
* **Hourly Buying Patterns:** Discovered dual-peak order waves occurring during midday intervals and late-evening hours (around 18:00/6 PM), providing clear optimization windows for targeted marketing.
